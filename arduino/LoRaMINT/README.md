# LoRaMINT - Arduino Library

LoRaMINT is a general-purpose, long-range measurement system for use in STEM subjects in schools. This library is part of the overall system and serves as the basis for Arduino programs of the system's sensor components.

## Parts of the Library
The library consists of the files 

 * keywords.txt
 * LoRaMINT.cpp
 * LoRaMINT.h

.
